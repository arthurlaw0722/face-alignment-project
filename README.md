# Face Alignment Model (Computer Vision)

This project develops a machine learning model to predict facial landmarks (face alignment) using computer vision techniques in Python.

## Overview
The model takes facial images as input and predicts key landmark points on the face, such as eyes, nose, and mouth positions.

## Key Features
- Image preprocessing and normalisation
- Facial landmark prediction using regression-based models
- Exploration of alternative approaches including CNN-based models
- Visualisation of predicted facial landmarks

## Results
- Successfully predicted facial keypoints on test images
- Compared performance between different modelling approaches
- Visualised model outputs for evaluation

## Technologies Used
- Python
- NumPy
- OpenCV (if used)
- PyTorch (if used)
- Matplotlib

## How It Works
1. Load and preprocess image data  
2. Train model to predict landmark coordinates  
3. Evaluate prediction accuracy  
4. Visualise predicted landmarks
